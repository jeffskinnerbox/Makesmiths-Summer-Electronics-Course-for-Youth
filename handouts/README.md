
Student-facing materials to be printed and handed out in class for the **Summer Electronics Course for
Youth** (Makersmiths course code **ELL-W101**). These are quick-reference aids students keep at the bench,
not lesson plans for the instructor. They are auxiliary materials — *not* outputs of the `.claude/skills/`
pipeline and they do *not* follow the appendix convention — but they must stay factually consistent with
the syllabus, BOM, and lesson plans (component names, class numbering, course code).


| File | Purpose |
|:-----|:--------|
| `cheatsheet.md` | Per-class student cheat sheet (starting with Class 1) — condenses the key ideas into at-a-glance form, e.g. the water-vs-electricity analogy mapping pressure/flow to voltage/current and their measurement units. |
