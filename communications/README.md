
Hand-written outreach documents for the **Summer Electronics Course for Youth**
(Makersmiths course code **ELL-W101**). These are addressed to parents and students rather than used to
teach the class. They are auxiliary materials — *not* outputs of the `.claude/skills/` pipeline and they do
*not* follow the appendix convention — but they must stay factually consistent with the syllabus, BOM, and
lesson plans (class dates, location, course code).


| File | Purpose |
|:-----|:--------|
| `letter-to-parents.md` | Introductory letter from the instructor to parents — explains the course's goal (giving students intuition for how electricity works), its hands-on nature, and points parents to the syllabus. Sent before the course begins. |
| `registration.md` | Event/registration details for the Makersmiths event listing: reservation ID and passcode, start date and time, duration (6 consecutive Tuesdays), and the Leesburg, VA location.  It also include student/paret name, contact information, etc. **NOTE:** This file is not posted (aka surpressed by `.gitignore`) for privacy reasons. |
